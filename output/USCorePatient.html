US Core Patient

Page standards status: Informative

Previous Page - US Core Organization

The US Core 3.1.1 Patient or US Core 6.1.0 Patient resource pdex-32: *SHALL** be used to express a member's demographic information. §1

The essential fields (Must Support or Cardinality greater than 0..*) to be provided in the US Core 3.1.1 Patient or US Core 6.1.0 Patient resource are:

The Member Number will be entered in the identifier field. The code MB SHALL be used to identify the member identifier. §pdex-33 See the example below:

{
      "use" : "member",
      "type" : {
        "coding" : [
          {
            "system" : "http://hl7.org/fhir/v2/0203",
            "code" : "MB",
            "display" : "Member Number"
          }
        ],
        "text" : "Health Plan Member Number"
      },
      "system" : "http://plan.healthplan.com",
      "value" : "987654321"
    }
