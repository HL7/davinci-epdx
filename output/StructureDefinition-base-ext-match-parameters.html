Extension: Member-Match Input Patient Parameter

Official URL: http://hl7.org/fhir/us/davinci-pdex/StructureDefinition/base-ext-match-parameters Version: 2.2.0
Standards status: Trial-use Maturity Level: 2 Computable Name: MatchParameters

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

Input Patient Parameter supplied for the individual member match (Patient Demographics).

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Group.member.entity

Usage info

Usages:
