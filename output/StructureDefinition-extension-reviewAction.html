Extension: ReviewAction

Official URL: http://hl7.org/fhir/us/davinci-pdex/StructureDefinition/extension-reviewAction Version: 2.0.0
Draft as of 2024-01-05 Computable Name: ReviewAction

The details of the review action that is necessary for the authorization.

Context of Use

This extension may be used on the following element(s):

  • Element ID ExplanationOfBenefit.item.adjudication

Usage info

Usage:

  • Use this Extension: PDex Prior Authorization
  • Examples for this Extension: ExplanationOfBenefit/PDexPriorAuth1
